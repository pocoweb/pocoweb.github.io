# 联系人(Contacts)数据接收协议

# 一、简介

# 接收地址

接收请求入口：

https://oct.pocoiq.cn/com.snowplowanalytics.iglu/v1

# HTTP 请求头

参数 值
Content-Type application/json; charset=utf-8

# 请求参数

参数 值 备注
schema iglu:com.pocoweb/contact/jsonschema/1-0-1 提交数据的JSON Schema
aid <填写组织下某应用的app_id> 组织下某应用的app_id，可以用来区分不同的用户组，但目前 (v0.8) 联系人功能并不区分，如果有此需求欢迎探讨。

# 完整地址

https://oct.pocoiq.cn/com.snowplowanalytics.iglu/v1?schema=iglu%3Acom.pocoweb%2Fcontact%2Fjsonschema%2F1-0-1&aid=YOUR_APP_ID

# 二、请求示例

请注意：其中 YOUR_APP_ID 是一个 App ID，通常可以通过 PocoIQ 后台获得，任意已集成的应用的App ID都可以使用。

# Curl

# 三、数据详细Schema

# 编码

  • 提交内容必须使用UTF8编码。

# 必填字段

  1. 下列字段必须填写，没有内容填空字符串。
    • email
    • mobile_number
    • user_id
    • changed_on
  2. 对于同一个联系人，系统会根据changed_on来确定最新版本的信息
  3. email / mobile_number / user_id 中，必须有一个有内容，否则将无法归并到正确的联系人记录。

# 完整Schema

字段名称 类型 长度 必填 备注
first_name string [..255] 否 名字
last_name string [..255] 否 姓氏
name string [..255] 否 完整姓名 (如不填，系统会根据first_name和last_name字段自动生成此字段内容)
salutation string [..255] 否 称呼
gender string [1] 否 性别 (取值: M - 男性; F - 女性)
birthday date [10] 否 生日（格式：YYYY-mm-dd，示例：2019-01-01）
email email [..255] 是 电子邮件
fax_number string [..255] 否 传真号码
mobile_number string [..255] 是 手机号码
telephone_number string [..255] 否 座机号码
country_region string [..255] 否 国家/地区
province string [..255] 否 省份
city string [..255] 否 城市
street_address string [..255] 否 街道地址
postal_code string [..255] 否 邮政编码
department string [..255] 否 部门
job_title string [..255] 否 职位
company_name string [..255] 否 公司名称
company_annual_revenue string [..255] 否 公司年收入
company_employee_count string [..255] 否 公司员工数
company_industry string [..255] 否 公司所属行业
company_website string [..255] 否 公司网址
user_id string [..255] 是 网站用户ID
wechat_open_id string [..255] 否 微信 Open ID
user_name string [..255] 否 网站用户名
description string [..255] 否 描述
registration_time date-time [..255] 否 网站用户注册时间，格式示例：2019-01-01T00:00:00Z
created_on date-time [..255] 否 记录创建时间，格式示例：2019-01-01T00:00:00Z
changed_on date-time [..255] 是 记录修改时间（修改时间最晚的记录会替代更早的记录），格式示例：2019-01-01T00:00:00Z

# 注意

# 日期格式

日期需要使用 ISO 8601 规范，形如 yyyy-MM-ddTHH:mm:ssZ 的格式。

例如在 C# 中可以使用如下方法。

DateTime.UtcNow.ToString("yyyy-MM-ddTHH:mm:ssZ")
